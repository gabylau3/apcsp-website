Linux Quick Reference Page

Name Description Example
cd Change the directory sp2a@rpi12:~ $ cd scripts
sp2a@rpi12:~/scripts $ cd
sp2a@rpi12:~ $
pwd Print the current working directory sp2a@rpi12:~ $ cd scripts
sp2a@rpi12:~/scripts $ cd
sp2a@rpi12:~ $
ls List directory contents sp2a@rpi12:~ $ ls
play work
mkdir Make a new directory sp2a@rpi12:~ $ mkdir website
sp2a@rpi12:~ $ ls
play work website
rm -rf Remove directory and all files in directory (no warning). sp2a@rpi12:~ $ rm -rf play
sp2a@rpi12:~ $ ls
work website
cp Copy files and directories sp2a@rpi12:~ $ cp /etc/passwd password
sp2a@rpi12:~ $ ls
work website password
mv Move files and directories sp2a@rpi12:~ $ mv password work
sp2a@rpi12:~ $ cd work
sp2a@rpi12:~/work $ ls
password
echo Display a line of tesk sp2a@rpi12:~ $ echo $HELLO
Hey there
nano Make a script/file sp2a@rpi12:~ $ nano myfirstbashscript
chmod Used to change permissions sp2a@rpi12:~ $ chmod +x filename
